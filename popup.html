Getting Started

Setup a shortcut that will open the command bar. Find the tab search extension and setup any shortcut that you like. Remember that some shortcuts reserved for Chrome (like Ctrl+T, or Ctrl+W) cannot be used. Ctrl+E is a great one since it's current binding isn't very useful (and also bound to Ctrl+K). Configure your shortcut here

Options

A search is performed by default after at least 3 characters have been typed. You can change this behavior by selecting the number of characters before performing a query below.

Hints

There are some things you should know about to help you out, they are listed below

The tab icon means that this result is from a currently open tab. Press enter when highlighted to focus on that tab
The tab with a plus means it is some sort of link and will open in a new tab if selected
If no search results are found in open tabs or bookmarks, the magnifying glass appears. Pressing enter will perform a Google search on the current term in a new tab
Some results will have additional options that appear as buttons in the corner. You can click on them to execute the operation
Ctrl+backspace
 Control+Backspace is a shortcut that will perform the additional option without needing to click on it.
Ctrl+Enter
 Control+Enter will perform a Google websearch on the term regardless of whether or not there are results
Esc
 Escape will close the command bar
